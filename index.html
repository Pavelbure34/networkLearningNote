day16.special IPv4 addresses

Directed Broadcast address

A address that host sends data to all the network on a specific network.
    Properties
  • The host portion is binary 1.
  • Example..
    If 172.16.0.0, broadcast address is 172.16.255.255.
  • It is disabled by default.
    They can be used for hacking attack!

Local Broadcast address

A address that host communicate with all the devices on a local network.
    Properties
  • addres is all binary 1s.
  • Example..
    255.255.255.255.
  • When it is used
    • host request IP address from DHCP server

    Key terms

  • Internet(WWW): a forest of thousands of routers.
  • Network: a connection between the two computer or devices.
    • It can be phones,printers, computers.
  • Network Protocol: a set of rules and conventions that governs networking.
    It is similar to language in conversation.
    and provides standards.
  • Router: a device that moves the data among different interfaces.
    • Distance is the most important factors for routing.
    • It can give wired or wireless connectivity where phones, tablets, laptops can connect to.
  • Switch: a device that moves the data among different interfaces.
    • Distance is the most important factors for routing.
    • it gives you multiple ports you can connect to.
  • IP address: Internet Protocol address = layer3(network) address assigned by admin.
    • Similar to pointer in C++ or home address .
    • Resides in Layer3 of OSI model
    • It is not hard coded like MAC address
    • Used to identify unique devices
    • Ipv4 address helps identify unique device in IP address .
        It has
      • 4 octet length(0~255)
  • DNS( Domain Name Service)= a system changing IP addresses into names.
  • OSI( Open System Interconnection) model governs how devices communicate to each other.
  • TCP(Transmission Control Protocol),UDP(User Datagram Protocol):
  • Data Encapsulation=During the host communication,data from one device is encapsulated with each layers packet.
  • Data Deencapsulation>=During the host communication,data from one device is De-encapsulated and sent to the other device.

  • You can track down paths and time with ping and tracert commands in windows cmd.
